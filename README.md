# Pattern Problems
- These problems will help you develop your logic building skills needed for DSA.

n = 4

### Pattern 1: Box
````
****
****
****
****
````

java:

````java
for(int i = 0; i<n; i++) {
    for(int j = 0; j<i+1; j++){
        System.out.print(" ");
    }
    for(int k = 2*(n - i) - 1; k > 0; k--) {
        System.out.print("*");
    }
    System.out.println();
}
````

### Pattern 2: Star Staircase
````
*
**
***
****
````

java:

````java
for (int i = 0; i<n; i++) {
    for (int j = i+1; j > 0; j--) {
        System.out.print("*");
    }
    System.out.println();
}
````
